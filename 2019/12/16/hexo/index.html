部署到远端服务器

免费部署到GitHub

  1. 创建新仓库：

    登录github，点击右上角十字（添加）图标，选择 new repository

    Repository name有特殊要求：xxx.github.io xxx为Owner，你的用户名

    Repository name也是你将来远程部署的网站地址

  2. 安装hexo部署插件

    npm i --save hexo-deployer-git

  3. 在博客文件夹中找到 _config.yml 配置文件 – 打不开的话下载一个notepad++

  4. 在 _config.yml 文件中找到 #Deployment，并进行配置

    5.输入 hexo d 进行远端部署

​ 如果是第一次使用github需要设置用户名和邮箱，不然会报错：

之后就会叫你输入账号密码，如果输入密码的时候的没有显示的话，不要怕，他有在输入只是没显示，输完密码直接ENTER就行
