Call原理(apply唯一不同是以数组方式传入参数)

context为第一个参数(Bar)，args为其余参数(‘bar’)，我们来逐步分析：

  1. context = context||window; 判断是否传入参数，没有传入参数默认指向window
  2. context.fn = this; 给context添加了属性fn，这时this为使用者Foo。
  3. const result = context.fn(...args); 把其他参数传入fn，并赋值给第三方，因为后续要删除fn
  4. delete context.fn; 删除fn
  5. return result; 返回context.fn(…args);这时候context

其实说白了就是给目标者(Bar)添加了一个fn方法，fn方法为使用者并且传入了其余参数(‘bar’)，最后将fn return出来。

new构调用

new会做什么

当使用new来调用函数时（发生构造函数调用时），JavaScript会做以下工作：

​ 1.创建一个全新的对象

​ 2.把新对象的__proto__属性指向函数的prototype属性

​ 3.this指向这个新对象

​ 4.如果函数没有返回其他对象，则返回新对象

对象的基本了解

JavaScript中的数据类型如下：(symbol为ES6新增，不讨论)

  • string

  • number

  • boolean

  • null

  • undefined

  • object

string,number,boolean,null,undefined是基本数据类型

object是复杂数据类型。object之所以叫复杂数据类型，是因为JavaScript中拥有一些对象子类型。也就是我们平时说的内置对象。内置对象有以下几个：(不要嫌这些分类、概念麻烦，但真挺有用)

+ String
+ Number
+ Boolean
+ Object
+ Function
+ Array
+ Date
+ RegExp
+ Error

可以看出基本数据类型除了null,undefined都有对应的内置对象。

所以引擎在需要的时候都会将字符串、数字、布尔类，这三种数据类型转换成对应的内置对象，通过new构造调用。

undefined,null只有文字形式，没有对应的构造形式(内置对象)。所以你就不能对null,undefined进行length,toString等操作。

当我们对字符串、数字、布尔字面量进行操作时，引擎会适时的将其使用new将字面量转换成对应的构造函数再进行操作。
